# 결재서 자동 작성 스킬

메일/견적서/계약서를 분석하여 그룹웨어 결재서 초안을 자동 생성합니다. 필드별로 복붙 가능한 형식으로 출력합니다.

## 워크플로우

### Phase 1: 유형 선택

사용자에게 결재서 유형을 질문합니다:

```
어떤 유형의 결재서를 작성할까요?

1. 계약서 품의
2. 일반 지출결의서
3. 시행품의서
4. 국외 외화 지출품의
```

사용자가 유형을 선택하면 Phase 2로 진행합니다.

### Phase 2: 자료 수집

자료를 수집하는 방법을 확인합니다:

**방법 A: Gmail 검색**
1. 관련 메일 키워드를 물어봅니다 (예: 거래처명, 견적서 등)
2. `gmail_attachment.py search "키워드"` 실행하여 메일 목록 표시
3. 사용자가 메일을 선택하면 `gmail_attachment.py fetch <message_id> --output /tmp/approval-draft/` 실행
4. 다운로드된 본문(`/tmp/approval-draft/body.txt`)과 첨부파일(PDF/이미지)을 Read 도구로 분석

```bash
python3 .claude/skills/approval-draft/scripts/gmail_attachment.py search "키워드" --limit 5
python3 .claude/skills/approval-draft/scripts/gmail_attachment.py fetch <message_id> --output /tmp/approval-draft/
```

**방법 B: 파일 직접 제공**
사용자가 파일 경로를 제공하면 Read 도구로 직접 분석합니다.

### Phase 3: 필드 채움 + 확인

추출한 정보로 해당 유형의 필드를 자동으로 채웁니다.

**고정값 (자동 삽입):**
- 기안자: 김규림
- 회계단위: 그란데클립 에프앤비 코리아
- 시행자: 경영관리팀 (지출결의서인 경우)
- 프로젝트: PJ) Newmix
- 부서: Newmix Korea

**채울 수 없는 필드가 있으면** 사용자에게 질문합니다.
**모든 필드를 채운 후** 결과를 보여주고 수정할 부분이 있는지 확인합니다.

### Phase 4: 복붙용 출력

필드별로 `[필드명] 값` 형식으로 출력합니다. 그룹웨어에서 각 필드에 복붙할 수 있도록 정리합니다.
