# 뉴믹스 마케팅 대시보드 디자인 가이드

## 1. 디자인 철학

"안전한 신뢰, 따뜻한 기록" - '꿈이 자라는 뜰' 스타일가이드 기반

데이터를 차갑게 나열하는 대시보드가 아니라, 따뜻하고 포용적인 느낌을 유지하며 데이터 기반 의사결정을 돕는 도구입니다.

**톤앤매너**: 차분한, 정돈된, 따뜻한, 신뢰할 수 있는

## 2. 컬러 팔레트

로고 핵심 색상 기반으로 구성된 따뜻하고 친근한 팔레트입니다.

### 주요 색상

| 역할 | 색상명 | HEX | 사용처 |
|------|--------|-----|--------|
| Primary | 올리브 그린 | `#849A6C` | 핵심 버튼(CTA), 활성 탭, 주요 아이콘, 대만 차트 |
| Secondary | 웜 오렌지 | `#F27D42` | 보조 버튼, 태그, 일본 차트 |
| Accent | 소프트 옐로우 | `#F6C06B` | 하이라이팅, 알림, 강조 요소 |
| Sub | 틸 그린 | `#4A897F` | 헤더/푸터 배경, 보조 아이콘, 중국 차트 |

### 배경 및 중립 색상

| 역할 | 색상명 | HEX | 사용처 |
|------|--------|-----|--------|
| Background | 웜 아이보리 | `#FEFBF6` | 전체 페이지 배경 (순백색보다 눈이 편안) |
| Text | 다크 그레이 | `#333333` | 본문 텍스트 (순검정보다 부드러움) |
| Text Sub | 미디엄 그레이 | `#888888` | 라벨, 메타 정보, 플레이스홀더 |
| Border | 라이트 그레이 | `#EAEAEA` | 카드 테두리, 구분선, 비활성 배경 |

### 차트 색상 조합

```
#849A6C, #F27D42, #F6C06B, #4A897F, #A8DADC
```

국가별 매핑:
- 일본: `#F27D42` (웜 오렌지) - 가장 높은 매출 비중, 시각적 강조
- 대만: `#849A6C` (올리브 그린) - Primary 컬러, 안정적 톤
- 중국: `#4A897F` (틸 그린) - 성장 추세를 표현하는 색상

## 3. 타이포그래피

### 폰트 체계

| 용도 | 폰트 | 사이즈 |
|------|------|--------|
| H1 (페이지 제목) | NanumSquareRound Bold | 32px |
| H2 (섹션 제목) | NanumSquareRound SemiBold | 24px |
| H3 (카드 제목) | NanumSquareRound SemiBold | 18px |
| Body (본문) | Pretendard Regular | 16px (자간 -0.2px, 행간 1.7배) |
| Label (라벨) | Pretendard Medium | 14px |

### 웹 폰트 로드

```css
@import url('https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/variable/pretendardvariable.min.css');
@import url('https://hangeul.pstatic.net/hangeul_static/css/nanum-square-round.css');
```

## 4. 레이아웃 및 컴포넌트

### 전체 구조

```
+--------------------------------------------------+
| [N] 뉴믹스 마케팅 대시보드           [날짜 필터]  |
|     국가별 마케팅 예산 배분 의사결정               |
+--------------------------------------------------+
| [🇯🇵 일본 카드]  [🇹🇼 대만 카드]  [🇨🇳 중국 카드]  |
|  매출 + 변동률     매출 + 변동률     매출 + 변동률  |
+--------------------------------------------------+
| [도넛 차트]  |  [라인/막대/누적 영역 차트]          |
|  매출 비중   |   일간 매출 추이                     |
+--------------------------------------------------+
| [총매출] [일평균] [최고매출일] [1위국가]            |
+--------------------------------------------------+
| [상세 데이터 테이블]                               |
+--------------------------------------------------+
```

### 카드 (Cards)

```css
border-radius: 12px;
box-shadow: 0 2px 20px rgba(0, 0, 0, 0.04);
border: 1px solid #EAEAEA;
```

- 메트릭 카드: 국가별 색상 그라디언트 배경 (`linear-gradient(135deg, {color}15, {color}08)`)
- 요약 카드: 흰색 배경 + 상단 컬러 보더 (`border-top: 3px solid {color}`)
- 차트 카드: 흰색 배경 + 라이트 그레이 보더

### 메트릭 카드 구성

```
+------------------------------+
| 🇯🇵 일본              54.1% |  <- 국기 + 국가명 + 매출비중 뱃지
|                              |
| 871,178원                    |  <- NanumSquareRound Bold, 큰 숫자
|                              |
| ▲ 2.1% 전일  ▲ 10.6% 전주   |  <- 변동률 (올리브=상승, 오렌지=하락)
+------------------------------+
```

### 버튼

- **Primary**: 올리브 그린(`#849A6C`) 배경 + 흰색 글씨
- **Secondary**: 라이트 그레이 배경 또는 올리브 그린 테두리
- **border-radius**: 8px
- **hover**: 0.15s 트랜지션

## 5. 아이콘

- 스타일: Lucide Icons / Feather Icons (둥근 라인 스타일)
- 국가 플래그: 🇯🇵 🇹🇼 🇨🇳

## 6. 인터랙션

- **Hover**: 카드/버튼에 0.15s ease 트랜지션 (색상 밝아지거나 그림자 짙어짐)
- **차트**: spline 곡선으로 부드러운 라인, hover 시 통합 툴팁
- **로딩**: Streamlit 기본 스피너 사용

## 7. 레퍼런스

이 디자인 가이드는 아래 두 가지를 결합하여 작성되었습니다:

1. **'꿈이 자라는 뜰' 스타일 가이드** - 컬러 팔레트, 폰트, 컴포넌트 규격
2. **모던 대시보드 UI 레퍼런스** - 카드 레이아웃, 그라디언트 메트릭, 넓은 여백 구조
