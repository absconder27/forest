# 뉴믹스 마케팅 대시보드 PRD

## 1. 제품 개요

| 항목 | 내용 |
|------|------|
| **제품명** | 뉴믹스 마케팅 대시보드 |
| **사용자** | 마케팅팀 전체 |
| **핵심 목적** | 국가별/채널별 마케팅 예산 배분 의사결정 |
| **대상 국가** | 일본, 대만, 중국 |
| **구현 형태** | 웹 대시보드 (Streamlit) |
| **업데이트 주기** | 매일 (전날 데이터 반영) |

## 2. 현재 문제점

1. **데이터 분산**: 국가마다 언어/플랫폼이 달라 데이터가 흩어져 있음
2. **트래킹 부재**: 인플루언서 포스팅이 올라가도 체계적으로 정리하지 않음
3. **상관관계 불명확**: 어떤 콘텐츠가 터져서 매출로 이어졌는지 파악 불가

## 3. 데이터 소스

| 데이터 | 소스 | 수집 방법 |
|--------|------|-----------|
| 국가별 매출 | PayHere | 서원님이 Gmail로 보고 (일간/주간) |
| SNS 포스팅 | Instagram, TikTok, YouTube, X, 블로그 | 태그/해시태그/멘션 기반 |
| 방문객 데이터 | PayHere (추정) | 매출 보고에 포함 |

## 4. 핵심 KPI

1. **국가별 매출 & 매출 비중** - 일본/대만/중국 매출 추이 및 비중 변화
2. **국가별 포스팅 수 & 반응** - 저장수, 좋아요, 조회수
3. **콘텐츠 <> 매출 스파이크 상관관계** - 타임라인 비교

## 5. 판단 기준

- 포스팅 반응(저장수 등) -> 해당 일자/국가 매출 변동 비교
- 방문객 비율(%)을 핵심 판단 지표로 활용

## 6. 메인 화면 구성

**통합 대시보드** (매출 + 콘텐츠 + 방문객을 하나의 화면에)

### 섹션 A: 국가별 매출 현황
- 일본/대만/중국 매출 비중 파이차트
- 국가별 매출 추이 라인차트 (일간)
- 전일/전주 대비 변동률

### 섹션 B: 콘텐츠 트래커
- 국가별 최근 인플루언서 포스팅 목록
- 포스팅별 반응 지표 (저장, 좋아요, 조회수)
- 플랫폼별 분류 (Instagram, TikTok, YouTube, X, 블로그)

### 섹션 C: 콘텐츠-매출 상관관계
- 타임라인 오버레이: 포스팅 시점 + 매출 스파이크
- 국가별 필터링

## 7. MVP 로드맵

### Phase 1 (MVP): 국가별 매출 현황
- Gmail에서 서원님 매출 보고 데이터 파싱
- 국가별 매출 비중 시각화
- 일간/주간 추이 차트
- **기술**: Streamlit + Gmail API (GWS 연동 활용)

### Phase 2: 콘텐츠 트래커
- SNS 포스팅 수동/반자동 입력 시스템
- 국가별 포스팅 반응 정리

### Phase 3: 상관관계 분석
- 매출 타임라인 + 콘텐츠 타임라인 오버레이
- 스파이크 감지 및 매칭

## 8. 기술 구현 계획 (MVP)

### 사용 기술
- **프레임워크**: Streamlit (Python 웹 대시보드)
- **데이터 수집**: Gmail API (GWS 연동 - 이미 인증 완료)
- **데이터 저장**: Google Sheets 또는 로컬 CSV
- **시각화**: Plotly

### 핵심 파일
- `00-system/02-scripts/gws/gws_auth.py` - GWS 인증 (재사용)
- `dashboard/app.py` - Streamlit 메인 앱
- `dashboard/data_fetcher.py` - Gmail 매출 데이터 파싱
- `dashboard/charts.py` - 차트 컴포넌트

### 구현 순서
1. Gmail API로 서원님 매출 보고 메일 조회 및 파싱
2. 국가별 매출 데이터 구조화
3. Streamlit 대시보드 UI 구성
4. 매출 비중 파이차트 + 추이 라인차트 구현
5. 매일 자동 업데이트 (cron 또는 수동 새로고침)

## 9. 검증 방법
- Gmail에서 실제 서원님 매출 보고 메일 1건을 파싱하여 데이터 추출 확인
- Streamlit 로컬 실행으로 차트 정상 렌더링 확인
- 마케팅팀 피드백 수집 후 Phase 2 진행 여부 결정
