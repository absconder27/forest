# PRD: Kyurim Kim 개인 웹사이트

## 1. 개요

**프로젝트명**: kyurimkim.com (도메인 미정)
**목적**: 김규림이라는 사람을 한눈에 보여주는 개인 웹사이트
**진입점**: Instagram @kyurimkim 프로필 최상단 링크
**대상**: 김규림에 대해 궁금한 모든 사람
**핵심 인상**: "개인인데 이렇게 창의적으로 웹사이트를 만들다니"

---

## 2. 디자인 컨셉

### 레퍼런스
- R.O. Blechman 공식 웹사이트 — 일러스트 기반 움직임이 있는 인트로

### 비주얼 스타일
- 김규림 셀프 일러스트 (미니멀 라인 드로잉, 흑백, 손맛 있는 선)
- 배경: 깨끗한 화이트
- 오브제: 실제 사물의 누끼컷 이미지 (사진 기반)
- 일러스트 캐릭터 + 실사 누끼의 조합이 핵심 비주얼 아이덴티티

### 톤
- 격식체
- 최소한의 텍스트, 비주얼 중심

---

## 3. 사이트 구조 (싱글 페이지)

### 3-1. 히어로 섹션: 저글링 애니메이션

화면 중앙에 김규림 캐릭터(라인 드로잉)가 오브제들을 저글링하고 있는 애니메이션.

**캐릭터**
- 기존 셀프 일러스트 사용 (손을 위로 올린 미니멀 라인 드로잉)
- 아래에 "kyurimkim" 핸드라이팅 텍스트

**저글링 오브제 (총 6개, 각각 클릭 가능)**

| # | 오브제 | 이미지 소스 | 클릭 시 이동 |
|---|--------|------------|-------------|
| 1 | 뉴믹스 커피 스틱 | 제품 누끼 (제공 완료) | 뉴믹스 공식 웹사이트 |
| 2 | 대표 도서 A | 교보문고에서 표지 누끼 | 해당 도서 교보문고 페이지 |
| 3 | 대표 도서 B | 교보문고에서 표지 누끼 | 해당 도서 교보문고 페이지 |
| 4 | 대표 도서 C | 교보문고에서 표지 누끼 | 해당 도서 교보문고 페이지 |
| 5 | 노트 (회중일기) | 제공 완료 | 뀰로그 (blog.naver.com/absconder) |
| 6 | 헤드폰 (Nothing) | 제공 완료 | 김규림의 소비예찬 YouTube |

**애니메이션 동작**
- 페이지 로드 시 캐릭터가 오브제를 저글링하는 모션 시작
- 오브제들이 공중에서 순환하며 떠다님
- 각 오브제에 hover 시 살짝 확대 + 어디로 연결되는지 힌트 (툴팁 또는 라벨)
- 클릭 시 해당 링크로 이동 (외부 링크는 새 탭)

## 6. 핵심 요구사항

1. **모바일 퍼스트** — Instagram 프로필 링크에서 유입되므로 모바일 경험이 최우선
2. **빠른 로딩** — 이미지 최적화 필수 (누끼컷 WebP 변환)
3. **인터랙티브** — 저글링 애니메이션이 사이트의 첫인상을 결정
4. **미니멀** — 불필요한 요소 없이, 오브제와 캐릭터만으로 소통
5. **외부 링크 중심** — 사이트 자체는 허브 역할, 상세 콘텐츠는 외부로 연결

## 7. 개발 단계

### Phase 1: 에셋 준비
- 캐릭터 일러스트 디지털화 (배경 제거, PNG/SVG 변환)
- 대표 도서 3권 선정 및 표지 누끼 준비
- 모든 외부 링크 URL 확정

### Phase 2: 정적 레이아웃
- 싱글 페이지 HTML/CSS 구조 작성
- 캐릭터 + 오브제 배치
- 연락처 섹션 구현
- 모바일 반응형 적용

### Phase 3: 애니메이션
- 저글링 모션 구현 (CSS 또는 JavaScript)
- 오브제 hover/클릭 인터랙션
- 로딩 최적화

### Phase 4: 배포
- 도메인 연결
- 호스팅 배포
- Instagram 프로필 링크 업데이트
